# Sensor Temperature and Humidity with NodeMCU

### Techs

-  Made with nodemcu, firebase and react 
- Plugins used in react:
  - Styled-components
  - Firebase
  - Axios
  - React-icons

### Some things that will be needed to run the project:
- NodeMCU - ESP8266
- Sensor DHT 11
- Some jumpers
- Resistor 220 ohms
- One LED
